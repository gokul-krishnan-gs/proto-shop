# ProtoShop

A simple and elegant frontend-only shopping cart application built using React, Context API, and React Router.

This project demonstrates how a complete shopping experience can be built without any backend, using only frontend technologies and localStorage.

## Features

* Browse products
* View product details
* Add items to cart
* Increase / decrease item quantity
* Remove items from cart
* Cart total calculation
* Checkout page (Under Construction)
* Fully responsive UI
* Frontend-only logic (No backend)

## Tech Stack

* React
* React Router
* Context API
* Tailwind CSS
* LocalStorage
* DummyJSON API (for products)
